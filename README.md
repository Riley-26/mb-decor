# MB Decor - Decorating Specialist

A bespoke website made for MB Decor, a Kent-based decorating business. The client wanted a modern, premium feeling website with a simple colour palette, reflecting their tidy and meticulous approach to their work. Built using Next.js for its server-side capabilities such as optimised load times and SEO.

## Architecture

The client wanted to keep the website as more of a brochure-style, as opposed to a conversion-optimised website. For this, I used Next.js to keep it running smoothly with content being rendered server-side (ideal for SEO). To work with Next.js, Typescript is my ideal standard for any web application that I make. This minimised compilation errors which was especially handy in my case, as a solo developer.

- Next.js
- Typescript
- TailwindCSS

## Key decisions

- To keep most of the content rendered server-side while playing smooth animations, I created a client-component which could be used to wrap around any piece of content needing an animation. This way, the content stayed server-rendered while keeping the animation utilities.
- As the client wanted a cool-looking site, many of the elements were designed for horizontal viewing (desktop). This required reshuffling of certain elements so that they remained responsive on mobile devices. Resulted in a dynamic experience between mobile and desktop.
- The client also wanted a new logo, so we needed to keep to consistent styling, colour and font between the logo and the website. We decided on the logo after the website was finished which was much easier than the other way. My client was very pleased with the final result, and the logo tied the business' aesthetic together.
